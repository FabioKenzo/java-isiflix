# ☕ Java IsiFlix - Meus Primeiros Passos

Este repositório é o registro da minha jornada inicial no mundo do Java. Aqui estão concentrados todos os fundamentos, testes e lógica que desenvolvi enquanto aprendia a sintaxe da linguagem.

## 🎓 Sobre o curso
Os códigos aqui presentes foram desenvolvidos durante o curso **Java IsiFlix**, ministrado pelo **Professor Isidro**. O curso foi fundamental para consolidar minha base em lógica de programação aplicada ao Java.

## 📂 Organização do Repositório

* **/aulas**: Contém exemplos práticos vistos durante as aulas para fixação de conceitos, além de diversos arquivos de teste (`Treinando.java`, `TesteStatus.java`, etc) onde explorei a linguagem por conta própria para validar o que estava aprendendo.
* **/exercicios**: Uma coletânea de resoluções de desafios da plataforma **Beecrowd (antigo URI)**. Aqui foquei em treinar algoritmos, manipulação de variáveis, estruturas condicionais e de repetição.

## 🛠️ O que eu pratiquei aqui:
- Entrada e saída de dados (`Scanner`).
- Operadores aritméticos e lógicos.
- Estruturas de decisão (`if/else`, `switch-case`).
- Estruturas de repetição (`for`, `while`, `do-while`).
- Manipulação de Arrays (Vetores) e estruturas de iteração como `ForEach`.

## 🚀 Próximos Passos
Após dominar essa base de sintaxe e lógica, já estou avançando para o estudo de **Programação Orientada a Objetos (POO)** e aplicações mais complexas.
